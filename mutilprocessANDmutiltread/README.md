```
1.忘记释放锁
2.资源竞争导致死锁，发生在两个线程之间的
```

futures   
可以像semaphore那样控制并发线程的数量  
可以在一个主线程中获取某一个线程的状态或者一个任务的状态，以及返回值  
当一个子线程完成的时候，主线程可以立即知道  
futures可以让多线程编程与多进程编程的代码规范和接口一致
