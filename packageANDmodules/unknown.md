```
10.8 读取位于包中的数据文件
注意事项：
事项：编程时任何I/O操作都必须使用绝对文件名
解释：
一则：一个包对解释器的当前工作目录几乎没有控
制权。由于每个模块包含有完整路径的__file__变量，
这弄清楚它的路径不是不可能，但它很凌乱。
二则：包通常安装作为.zip或.egg文件，这些文件
并不像在文件系统上的一个普通目录里那样被保存。
因此，你试图用open()对一个包含数据文件的归档
文件进行操作，它根本不会工作
推荐使用：
import pkgutil
data = pkgutil.get_data(__package__, 'somedata.dat')
```
